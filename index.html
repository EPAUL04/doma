Welcome to Doma!

Welcome to Doma!


Doma is a flashcard website aimed at practicing vocabulary while learning a language! Here's how to use it:


  1. First, fill out your dictionary to store your terms and translations/definitions, or head over to the library to find existing sets you would like to practice!
  2. Second, head over to the Flashcards section to get started! You can tap the card itself to turn it over, and hit the 'Next Card' button to go to the next card in the set!

-~✩~-
To select a color scheme, hover over the nav bar to see the dropdown menu with different options. Each icon represents a different color palette. When clicked, this theme will be applied to all pages within the site. The home page is the only page with this option, so come back to this page anytime you want to reset the color scheme. To check out the original sources of the icons, see the About page.

Click on the icons below to change the color scheme.


This website was designed for language learning, but works for other topics that you may want to use flashcards for. It is good for building and practicing vocab, but will not teach any important grammar rules, so it cannot be used to learn new languages or topics.
